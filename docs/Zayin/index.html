== External Services ==

The important part of an nmap scan:

=== Processes ===

  • vsftpd serves ftp.
  • openssh serves ssh.
  • postfix serves smtp.
  • bind serves domain.
  • apache2 serves http.
  • bitlbee
  • screen multiplexes between:
    • bash - for pretty much everything
    • irssi - for irc and im
    • mutt - for email
    • slrn - for news
    • snownews - for RSS aggregation
