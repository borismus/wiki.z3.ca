LegoBrick

page was renamed from Legopedia/Brick

The 1x1 lego brick is not a cube. The width:height ratio is 5:6. This is a useful relation to keep in mind when trying to create sturdy LEGO constructions. Additionally, one-third-height [wiki:LegoPlate plate] can be used between bricks.
